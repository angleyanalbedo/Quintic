# Quintic Web Client (MVP)

This is the MVP (Minimum Viable Product) for the Quintic Cam Editor, implementing a VDI 2143 compliant 5th-order polynomial generator.

## Features
- **Input:** Configure Master (deg) and Slave (mm) ranges.
- **Math Kernel:** 5th Order Polynomial (Rest-in-Rest).
- **Visualization:** Real-time S/V/A/J charts.
- **Export:** CSV generation for PLC import.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.
